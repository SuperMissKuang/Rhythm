Overview

Rhythm is a menstrual cycle tracking application that allows users to notify trusted contacts (partners, family members, friends) about cycle phase changes via SMS. This feature helps users communicate their needs during different phases of their cycle.

SMS Feature Rules

If you use the notification feature:

  • Only add contacts who would expect to receive these messages
  • Respect when contacts opt-out by replying STOP
  • Don't use it for spam or harassment
  • Message frequency varies based on your cycle (typically 1-2 per month)
  • Standard message and data rates may apply to recipients

Message Content

Rhythm sends only pre-defined cycle notification messages. All messages:

  • Contain only menstrual health updates
  • Include opt-out instructions
  • Do NOT contain sexual content, hate speech, or references to alcohol, firearms, tobacco, or marijuana
  • Do NOT contain promotional or marketing content

Health Disclaimer

Rhythm is not medical advice. Cycle predictions are estimates. Don't use this app for contraception or medical decisions. Consult a healthcare provider for medical concerns.
