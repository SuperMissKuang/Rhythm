SMS Opt-In & Consent Process Documentation

Overview

Rhythm is a menstrual cycle tracking application that allows users to notify trusted contacts (partners, family members, friends) about cycle phase changes via SMS. This feature helps users communicate their needs during different phases of their cycle.

Detailed Opt-In Process

1User Selects Contact

The app user navigates to "Manage Contacts" and taps "Add Contact". Instead of manual entry, they can tap "Choose from Contacts" to select from their device's native contact list.

Native iOS contact picker integration
⚠️ Important: Contact selection happens within the user's own device. The app does not upload or store any contact information until the user explicitly saves it.

2Opt-In Confirmation Dialog

After selecting a contact, the app shows a confirmation dialog asking the user if they want to send an opt-in request. This dialog:

  • Displays the contact's name and phone number
  • Shows a preview of the exact SMS message that will be sent
  • Clearly warns that notifications can only be sent after the recipient replies "YES"
  • Allows the user to cancel without sending anything
Add Contact form with "Choose from Contacts" button
Opt-in confirmation dialog with message preview

User must explicitly tap "Send Opt-In Request" for the SMS to be sent. This prevents accidental opt-in requests.

3SMS Opt-In Request Sent

When the user confirms, the following SMS is sent to the recipient:

[User Name] is inviting you to receive her cycle updates via Rhythm app. Reply YES to opt-in. Reply STOP to decline. Msg frequency varies (approx 1-2/month). Msg & data rates may apply.

Message Components:

  • "[User Name]" - Personalized sender identification
  • "Rhythm app" - Clear app/business identification
  • "Reply YES to opt-in" - Clear call-to-action for consent
  • "Reply STOP to decline" - Easy opt-out method
  • "Msg frequency varies (approx 1-2/month)" - Message frequency disclosure
  • "Msg & data rates may apply" - Standard carrier disclaimer

4Recipient Replies YES

The recipient must reply "YES" to the SMS to confirm they want to receive cycle notifications. Only after this explicit consent will the app mark the contact as "Confirmed" and allow notifications to be sent.

Note: Recipients can reply "STOP" at any time to opt-out. The app will respect this and stop sending notifications to that contact.

5Contact Status Tracking

The app tracks the opt-in status of each contact:

  • Pending - Opt-in request sent, awaiting recipient's response
  • Confirmed - Recipient replied YES, can receive notifications
  • Declined - Recipient replied STOP, will not receive notifications
Contact list showing "Pending" status badge with "Send Reminder" button

Only "Confirmed" contacts will receive cycle notifications. The app filters out pending and declined contacts when sending notifications.
